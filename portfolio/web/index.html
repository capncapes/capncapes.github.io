My web development journey

Back in 2015, I decided to finally learn HTML after a few years in marketing had shown me my passion for martech. So I started with HTML and CSS before moving on to JavaScript and eventually Ruby.

In addition to HTML and CSS, I've learned languages like JavaScript, SQL, AMPscript, PHP, Liquid, and Ruby; content platforms like WordPress and Shopify; and frameworks like Timber/Twig, Vue.js, and AngularJS. I enjoy learning new frameworks, platforms, and web languages.

I owe my tech career to my wonderful wife, Jackie, who saw my interest early on and encouraged me to learn the skills I'd need to work in web development.
